# User Personas - 냠픽(Yumpick)

## Persona 1: 고객 (Customer)

- **이름**: 김민수 (대표 페르소나)
- **역할**: 식당 방문 고객
- **특성**:
  - 테이블에 앉아 태블릿으로 주문
  - 별도 로그인 없이 바로 사용
  - 메뉴를 탐색하고 장바구니에 담아 주문
  - 주문 후 상태를 확인하고 싶음
- **목표**: 빠르고 편리하게 음식 주문
- **불편사항**: 직원을 기다리는 시간, 복잡한 주문 과정
- **기술 수준**: 스마트폰/태블릿 기본 사용 가능

## Persona 2: 매장 관리자 (Admin)

- **이름**: 박서연 (대표 페르소나)
- **역할**: 매장 운영자/사장님
- **특성**:
  - PC 또는 태블릿으로 관리 화면 사용
  - 들어오는 주문을 실시간으로 확인
  - 메뉴 등록/수정, 테이블 관리 수행
  - 하루 종일 대시보드를 켜놓고 운영
- **목표**: 효율적인 매장 운영, 주문 누락 방지
- **불편사항**: 주문 놓침, 수기 관리의 비효율
- **기술 수준**: 기본적인 웹 사용 가능, IT 전문가 아님
