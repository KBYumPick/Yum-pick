### Q1: 주문 번호 생성 규칙
주문 번호(orderNumber) 자동 생성 방식을 선택해주세요.
- A) 날짜 기반 순번 (예: 20260331-001, 20260331-002)
- B) 매장별 순번 (예: #001, #002 - 매일 리셋)
- C) UUID 기반 짧은 코드 (예: A3F2, B7K9)

[Answer]: A

### Q2: 주문 상태 전이 규칙
주문 상태 변경 시 역방향 전이를 허용할까요?
- A) 단방향만 허용 (대기중 → 준비중 → 완료, 되돌리기 불가)
- B) 역방향 허용 (완료 → 준비중, 준비중 → 대기중 가능)

[Answer]: A

- A) 5회 실패 시 15분 잠금
- B) 10회 실패 시 30분 잠금
- C) 3회 실패 시 5분 잠금

[Answer]:A

메뉴 카테고리를 어떻게 관리할까요?
- A) 고정 카테고리 (관리자가 메뉴 등록 시 텍스트로 직접 입력)
- B) 카테고리 별도 CRUD (카테고리를 먼저 등록하고 메뉴에서 선택)
